Introducción

Este es un sitio web y a su ves una guia introductoria a las ecuaciones diferenciales que explicara que es una ecuacion diferencial y como se clasifica una ecuación diferencial mediante el tipo, orden y grado para despues puedan ver una tabla con ejemplos de ecuaciones diferenciales y unas instrucciones sobre como resolver ecuaciones diferenciales en el Software Geogebra.

¿Qué es una ecuación diferencial?

A continuacion pondremos 3 definiciones de una ecuación diferencial:

  1. Una ecuación diferencial es aquella ecuación que contiene derivadas o diferenciales.
  2. Orden de una ecuación diferencial es el de la derivada de mayor orden contenida en la ecuación.
  3. Grado de una ecuación diferencial es la potencia a la que está elevada la derivada de mayor orden, siempre y cuando la ecuacion diferencial esté dada en forma normal.

Podemos decir de una manera mas facil y entendible que una ecuacion diferencial es una ecuacion, solo que la ecuacion siempre debera contener una derivada y tiene varias caracteristicas que clasifican de manera mas exacta a una ecuacion diferencial.

clasificación

Como ya mencionamos antes una ecuacion diferencial se clasfica de varias maneras y esas son las siguientes:

  1. Tipo:
    • Ordinarias:

      La ecuación diferencial contiene derivadas de una o más variables dependientes con respecto a una sola variable independiente.

    • Parciales:

      La ecuación diferencial contiene derivadas parciales de una o mas variables dependientes con respecto a dos o más variables independientes.

  2. Orden:
    • Primer orden:

      F(x, y, y') = 0

    • Segundo orden:

      F(x, y', y'') = 0

    • Tercer orden:

      F(x, y, y', y'', y''') = 0

    • *
    • Orden n:

      F(x, y, y', ..., \(y^n\)) = 0

  3. Grado:
    • Lineales:
      • La variable dependiente y todas sus derivadas son de primer grado.
      • Cada coeficiente de Y y sus derivadas depende solamente de la variable independiente X.
    • No lineales:

      Las que no cumplen las propiedades anteriores.
